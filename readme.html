Installation & First Run

1 Install Dependencies (first time only) 
julia src/00_First_time_install_packages.jl
2 Select Cases to Run
Edit cases_to_run.yaml to list your cases: 
case_folders:
  - "Stanford_bunny"
  - "Wing_5_deg"
3 Run the Solver 
julia src/_RUN_CASE_.jl
âœ“ Results are saved in CASES/<case_name>/RESULTS/
Open .vtu files in ParaView for visualization.

Creating a New Case

  1. Create folder in CASES/ (e.g., CASES/My_Case/)
  2. Add your STL geometry file
  3. Copy config.yaml from an existing case and modify
  4. Add case name to cases_to_run.yaml

Key Config Parameters

Parameter Purpose
surface_resolution Mesh density (higher = finer, more memory)
flow.velocity Inlet velocity [m/s]
simulation.steps Total timesteps to run
num_levels Grid refinement levels (0 = auto)
